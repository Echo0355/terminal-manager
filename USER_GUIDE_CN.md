### 区域说明

| 区域 | 功能 |
|------|------|
| **活动栏** | 左侧图标栏（48px），包含切换侧边栏和打开设置两个按钮，始终为深色主题 |
| **侧边栏** | 项目资源管理器，可添加/删除/打开项目，宽度可拖拽调整（150-400px） |
| **标签栏** | 工作区标签页，支持新建、关闭、拖拽排序，分屏时显示面板数量徽章 |
| **面板标签栏** | 分屏模式下每个面板内部的二级标签栏，显示面板标题和关闭按钮 |
| **终端区** | xterm.js 渲染的终端，支持水平/垂直分屏、拖拽调整大小 |
| **状态栏** | 底部信息栏，左侧显示 Shell 名称和工作目录，右侧显示面板数、编码和主题切换 |

## 终端操作

### 打开新终端

- 点击标签栏的 **＋** 按钮
- 按 `Ctrl+T`
- 从菜单"终端 → 新建终端"

### 输入命令

直接在终端中输入命令，与系统终端使用方式相同。

### 复制粘贴

- **复制**：选中文本后自动复制到剪贴板，或按 `Ctrl+Shift+C`
- **粘贴**：按 `Ctrl+Shift+V` 或 `Ctrl+V`
- **右键菜单**：有选中内容时复制，无选中内容时粘贴

### 关闭终端

- 点击标签上的 **×** 按钮
- 按 `Ctrl+W`
- 从菜单"文件 → 关闭标签"

关闭时会弹出确认对话框。

### 可点击链接

终端输出中的 URL 会自动识别为可点击链接。

## 标签管理

### 新建标签

点击标签栏的 **＋** 按钮或按 `Ctrl+T`。

### 切换标签

- 点击标签
- 按 `Ctrl+Tab` 切换到下一个标签
- 按 `Ctrl+Shift+Tab` 切换到上一个标签
- 按 `Ctrl+1~9` 切换到指定位置的标签

### 拖拽排序

标签支持拖拽重排序。按住标签拖动到目标位置即可。

### 分屏徽章

当标签内有多个面板时，标签上会显示面板数量的蓝色徽章，方便识别分屏状态。

### 关闭标签

- 点击标签上的 **×** 按钮
- 按 `Ctrl+W`

如果标签内有多个面板，会先确认再关闭。

## 分屏布局

### 水平分屏（左右）

- 按 `Ctrl+Shift+D`
- 从菜单"终端 → 水平分屏"

当前面板会左右分裂为两个面板，新面板自动获得焦点。

### 垂直分屏（上下）

- 按 `Ctrl+Shift+Alt+D`
- 从菜单"终端 → 垂直分屏"

当前面板会上下分裂为两个面板。

### 调整面板大小

鼠标悬停在面板之间的分割线上（4px 宽），光标变为 ↔ 或 ↕ 时拖拽调整。

分割线在悬停和拖拽时会高亮显示。拖拽时会实时更新布局，释放鼠标后自动归一化确保尺寸总和为 100%。

### 焦点切换

按 `Alt+方向键` 在相邻面板间移动焦点：

| 快捷键 | 功能 |
|--------|------|
| `Alt+←` | 焦点移到左侧面板 |
| `Alt+→` | 焦点移到右侧面板 |
| `Alt+↑` | 焦点移到上方面板 |
| `Alt+↓` | 焦点移到下方面板 |

获得焦点的面板会有蓝色边框高亮。

### 关闭面板

- 按 `Alt+Shift+W`
- 点击面板标签栏的 **×** 按钮

如果只剩一个面板，关闭面板会关闭整个标签。

## 拖拽操作

终端管理器支持 VS Code 风格的拖拽操作。

### 拖拽面板

拖拽面板标签可以在同一标签内重新排列，或移动到其他标签中。

### 拖拽到边缘

将面板标签拖拽到目标面板的边缘区域时，会触发分屏操作：

| 拖拽位置 | 效果 |
|----------|------|
| 目标面板中心 | 替换目标面板 |
| 目标面板左侧 | 在目标左侧水平分屏 |
| 目标面板右侧 | 在目标右侧水平分屏 |
| 目标面板上方 | 在目标上方垂直分屏 |
| 目标面板下方 | 在目标下方垂直分屏 |

拖拽时会显示方向指示器，提示释放后的位置。

### 拖拽到标签栏

将面板标签拖拽到顶部标签栏，会将该面板提取为新的独立标签。

## 项目管理

### 添加项目

1. 点击侧边栏顶部的 **＋** 按钮
2. 在弹出的系统目录选择对话框中选择项目目录
3. 项目会出现在侧边栏列表中

### 打开项目终端

- **双击项目名称**：在新标签中打开，工作目录为项目目录
- **点击 ▶ 按钮**：同上

### 删除项目

将鼠标悬停在项目上，点击 **✕** 按钮。会弹出确认对话框。

删除项目不会删除目录本身，只是从列表中移除。

### 侧边栏操作

- **调整宽度**：拖拽侧边栏右边缘（最小 150px，最大 400px）
- **折叠/展开**：点击活动栏的侧边栏图标

## 设置配置

按 `Ctrl+,` 打开设置对话框，或点击活动栏底部的齿轮图标。

### 外观设置

| 配置项 | 说明 |
|--------|------|
| 主题 | 深色或浅色主题，带可视化预览色块。**立即生效** |

### 终端设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 默认 Shell | 终端使用的 Shell 路径 | `powershell.exe` (Windows) / `/bin/zsh` (macOS) |
| 默认目录 | 新终端的默认工作目录 | 用户主目录 |
| 字体大小 | 终端字体大小 (8-32) | 14 |
| 滚动缓冲 | 终端历史记录行数 (100-100000) | 10000 |

> **注意**：终端设置（Shell、目录、字体大小、滚动缓冲）修改后需要重启应用才能生效。主题设置即时生效。

### 保存设置

修改后点击"保存"按钮。按 `Esc` 或点击对话框外部取消。

### 自动检测的 Shell

应用启动时会自动检测系统中可用的 Shell：

**Windows：**
- PowerShell（始终可用）
- CMD（从 `%COMSPEC%` 环境变量检测）
- Git Bash（`C:\Program Files\Git\bin\bash.exe`，需已安装 Git）
- WSL（`C:\Windows\System32\wsl.exe`，需已启用 WSL）

**macOS / Linux：**
- Zsh (`/bin/zsh`)
- Bash (`/bin/bash`)
- Sh (`/bin/sh`)
- Fish (`/usr/bin/fish` 或 `/opt/homebrew/bin/fish`)

只有检测到的 Shell 和配置文件中指定的默认 Shell 才允许使用。

## 快捷键参考

### 标签操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+T` | 新建标签 |
| `Ctrl+W` | 关闭当前面板（最后一个面板时关闭标签） |
| `Ctrl+Tab` | 下一个标签 |
| `Ctrl+Shift+Tab` | 上一个标签 |
| `Ctrl+1` ~ `Ctrl+9` | 切换到第 N 个标签 |

### 分屏操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+D` | 水平分屏（左右） |
| `Ctrl+Shift+Alt+D` | 垂直分屏（上下） |
| `Alt+Shift+W` | 关闭当前面板 |

### 焦点切换

| 快捷键 | 功能 |
|--------|------|
| `Alt+←` | 焦点左移 |
| `Alt+→` | 焦点右移 |
| `Alt+↑` | 焦点上移 |
| `Alt+↓` | 焦点下移 |

### 剪贴板

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+C` | 复制选中文本 |
| `Ctrl+Shift+V` / `Ctrl+V` | 粘贴 |

### 应用

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+,` | 打开设置 |
| `Esc` | 关闭对话框 |
