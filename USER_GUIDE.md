### 区域说明

| 区域 | 功能 |
|------|------|
| **菜单栏** | 应用菜单，包含文件、编辑、视图、终端、帮助 |

| **状态栏** | 显示当前 Shell、工作目录等信息 |

## 终端操作

### 打开新终端

- 点击标签栏的 **＋** 按钮
- 按 `Ctrl+T`
- 从菜单"终端 → 新建终端"

### 输入命令

直接在终端中输入命令，与系统终端使用方式相同。

### 复制粘贴

- **复制**：选中文本后自动复制，或按 `Ctrl+Shift+C`
- **粘贴**：按 `Ctrl+Shift+V`

### 关闭终端

- 点击标签上的 **×** 按钮
- 按 `Ctrl+W`
- 从菜单"文件 → 关闭标签"

关闭时会弹出确认对话框。

## 标签管理

### 新建标签

点击标签栏的 **＋** 按钮或按 `Ctrl+T`。

### 切换标签

- 点击标签
- 按 `Ctrl+Tab` 切换到下一个标签
- 按 `Ctrl+Shift+Tab` 切换到上一个标签
- 按 `Ctrl+1~9` 切换到指定位置的标签

### 关闭标签

- 点击标签上的 **×** 按钮
- 按 `Ctrl+W`

如果标签内有多个面板，会先确认再关闭。

## 分屏布局

### 水平分屏（左右）

按 `Alt+Shift+-` 或从菜单"终端 → 水平分屏"。

```
┌─────────────┬─────────────┐
│             │             │
│  Terminal 1 │  Terminal 2 │
│             │             │
└─────────────┴─────────────┘
```

### 垂直分屏（上下）

按 `Alt+Shift+=` 或从菜单"终端 → 垂直分屏"。

```
┌───────────────────────────┐
│        Terminal 1         │
├───────────────────────────┤
│        Terminal 2         │
└───────────────────────────┘
```

### 调整面板大小

鼠标悬停在面板之间的分割线上，光标变为 ↔ 或 ↕ 时拖拽调整。

### 焦点切换

按 `Alt+方向键` 在相邻面板间移动焦点。

获得焦点的面板会有蓝色边框高亮。

### 关闭面板

按 `Alt+Shift+W` 关闭当前焦点面板。

如果只剩一个面板，会关闭整个标签。

## 项目管理

### 添加项目

1. 点击侧边栏顶部的 **＋** 按钮
2. 在弹出的对话框中选择项目目录
3. 项目会出现在侧边栏列表中

### 打开项目终端

- **点击项目名称**：在新标签中打开，工作目录为项目目录
- **点击 ▶**：同上
- **点击 ⫼**：在当前标签水平分屏打开
- **点击 ⫟**：在当前标签垂直分屏打开

### 删除项目

将鼠标悬停在项目上，点击 **✕** 按钮。

删除项目不会删除目录本身，只是从列表中移除。

## 设置配置

按 `Ctrl+,` 打开设置对话框。

### 可配置项

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 默认 Shell | 终端使用的 Shell 路径 | `powershell.exe` (Windows) |
| 默认目录 | 新终端的默认工作目录 | 用户主目录 |
| 字体大小 | 终端字体大小 (8-32) | 14 |
| 滚动缓冲 | 终端历史记录行数 | 10000 |
| 主题 | 深色或浅色主题 | 深色 |

### 保存设置

修改后点击"保存"按钮。设置会在下次启动应用时生效。

### 常用 Shell 路径

**Windows:**
- PowerShell: `powershell.exe`
- CMD: `cmd.exe`
- Git Bash: `C:\Program Files\Git\bin\bash.exe`

**macOS:**
- Zsh: `/bin/zsh`
- Bash: `/bin/bash`
- Fish: `/opt/homebrew/bin/fish`

## 布局恢复

应用会自动保存布局状态，包括：

- 所有标签
- 分屏布局结构
- 每个面板的 Shell 和工作目录
- 侧边栏宽度

### 保存时机

布局状态会在以下操作后自动保存（带 500ms 防抖）：

- 新建/关闭标签
- 切换标签
- 分屏/关闭面板
- 拖拽分割线调整大小
- 调整侧边栏宽度

### 恢复逻辑

重启应用后：

1. 加载保存的布局状态
2. 按保存的 Shell 和工作目录重新创建终端
3. 恢复分屏布局结构
4. 恢复活动标签和焦点面板

### 注意事项

- 布局恢复不会恢复终端中正在运行的命令
- 不会恢复 Shell 的内存状态
- 如果保存的 Shell 路径无效，会使用默认 Shell
- 如果保存的工作目录不存在，会回退到用户主目录
